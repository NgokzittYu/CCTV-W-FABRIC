## 🎯 功能详细说明

### 1. 工单管理系统
**实现内容**:
- 完整的工单生命周期管理（创建 → 提交 → 确认 → 关闭）
- 工单列表展示（表格、筛选、搜索）
- 工单详情查看（状态历史、附件）
- 超期工单自动标记
- 状态流转可视化

**技术亮点**:
- 响应式设计，适配移动端
- 模态框交互，用户体验流畅
- 实时状态更新
- 链码集成，数据上链

### 2. 角色权限控制
**实现内容**:
- 三组织角色切换（Org1/Org2/Org3）
- 基于角色的动态权限控制
- 角色持久化（localStorage）
- 角色切换通知

**权限矩阵**:
| 操作 | Org1 | Org2 | Org3 |
|------|------|------|------|
| 创建工单 | ❌ | ✅ | ❌ |
| 提交整改 | ✅ | ❌ | ❌ |
| 确认整改 | ❌ | ✅ | ❌ |
| 查看工单 | 仅自己 | 全部 | 全部 |
| 导出审计 | ✅ | ✅ | ✅ |

### 3. 审计报告系统
**实现内容**:
- 审计报告导出（批次维度）
- 报告预览功能
- JSON 格式下载
- 报告签名生成
- 报告验证功能

**报告内容**:
- 报告元数据（ID、时间、签名）
- 批次信息（Merkle Root、事件数量）
- 证据事件列表
- 整改工单列表
- 验证信息

### 4. 自动触发工单
**实现内容**:
- 智能触发逻辑（批次事件 ≥5）
- 配置管理界面
- 规则引擎（违规等级、责任方、截止天数）
- 异步处理不阻塞
- 配置持久化

**触发规则示例**:
```json
{
  "violation_level": "high",
  "auto_assign_org": "Org1MSP",
  "default_deadline_days": 7
}
```
